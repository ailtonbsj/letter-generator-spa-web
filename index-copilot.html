Bem-vindo ao Gerador de Carta de Oposição

Use o formulário para preencher o modelo XML e gerar um PDF pronto para envio.

Como funciona

Preencha os campos no menu "Gerador PDF". O sistema substituirá padrões do tipo #FIELD_NAME# no XML e enviará o arquivo para conversão em PDF.

Gerador PDF — Preencha o formulário

Todos os campos são obrigatórios. O arquivo será enviado ao serviço externo para conversão em PDF.

Sobre

Aplicação de exemplo para gerar cartas a partir de um template XML e converter em PDF via API externa.
